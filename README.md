# 42-get_next_line

The purpose of this project is to code a function that returns the line read from a file descriptor

## Instructions

**Compiling the library**

```shell
$ cd 42-ft_get_next_line/ && make
```

**Using it in your code**

```C
#include "get_next_line.h"
```
